# Extract Calculator

A Python GTK application for extracting data from PDFs using pdfplumber.

## Features

- GTK-based UI
- PDF text extraction

## Installation

1. Clone the repository
2. Create virtual environment: `virtualenv venv`
3. Activate: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`

## Usage

Run `python -m src.extract_calculator.main`

## Development

Use basedpyright for LSP.

Run tests: `python -m unittest`
